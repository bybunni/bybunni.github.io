# bybunni.github.io

Personal portfolio site. Live at [TDerror.com](https://tderror.com).

## Setup
No build step required. Open `index.html` in a browser or push to GitHub Pages.

## Structure
```
index.html          Single-page portfolio
css/style.css       Styles and responsive layout
js/main.js          Scroll effects and mobile menu
assets/images/      Press photos and publication figures
```
